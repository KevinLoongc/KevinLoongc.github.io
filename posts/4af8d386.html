Ubuntu16.04使用Systemback制作系统镜像

本文最后更新于：2022年5月29日 上午

前言

最近公司在Ubuntu系统下进行项目开发，运行环境配置比较麻烦，而且需要在其它工控机上安装配置相同的环境，一台一台来进行安装不切实际，只能通过打包整个系统快速进行。这里记录使用Systemback制作系统镜像，进行系统环境快速配置。

Step 1: 下载安装systemback

安装完成后在Dash菜单中就能找到这个软件了。

Step 2: 制作sblive镜像文件

点击图标，输入管理员密码，打开后界面如下图所示

点击”Live system create”【创建live系统】

  1. “Working directory”：选择工作目录(一般维持默认就好)；
  2. “Name of the Live system”：填入镜像名称；
  3. “Include the user data files”：包含用户数据文件。可酌情勾选，勾选后生成的镜像会大很多。
  4. 配置好一切后，点击”Create new”创建镜像。
    然后等待创建完成即可。
  5. 选择创建完成的sblive文件，在窗口左下方选择写入设备，点击”Write to targe”即可。
    写入完成后，u盘就是有着镜像的启动盘了。

注意：如果系统语言为中文，后面进行U盘安装系统会出现乱码，可以选择第二项继续进行安装系统。

Step 3: 镜像安装，实现系统完整复刻

  1. 把u盘插入裸机，进入bios系统，选择优先启动u盘。启动后应该会有如下界面。

  2. 此时，选择第二项进行系统安装（图片是别人的，自己没截图，没差），填入各项信息，如下:

  3. 点击下一步进入分区界面。

  4. 选中你要安装的硬盘，然后点击Delete按钮

  5. 再次选择硬盘中的新分区

  6. 点击箭头进入下一步

再次选择刚才新建的分区
设置右侧的Mount point，将主硬盘挂载点选择到根目录**/，并且勾选上【Tansfer user configuration files】**,否则系统安装时不会导入个人文件，再次点击箭头进入下一步

点击右下角的next就可以继续了。如果你还想设置其他分区可以按照以上的步骤进行设置。
